缘起

最近发现知乎的登录界面改了，抓包看了看，果然登录方式也变了，会根据用户名字符串发送不同字段的POST。中午研究了一会，写了个登录知乎的工具类，可以自动处理Cookie，维持登录状态。目前仅实现登录和打开网页的功能。

依赖的第三方库

  • requests
  • BeautifulSoup
  • lxml 这个不装没关系，把__login()方法中的lxml改成html.parser即可。
