urllib和requests用于访问网络，BeautifulSoup用于解析网页，三者经常交叉使用，放一块儿介绍。这里只作简要备忘，更多内容须参阅官方文档。

requests

官方文档：http://www.python-requests.org/en/latest/

BeautifulSoup

官方文档：http://www.crummy.com/software/BeautifulSoup/bs4/doc/index.zh.html

安装时不要少输个数字：

pip install BeautifulSoup4

Beautiful Soup将复杂HTML文档转换成一个复杂的树形结构,每个节点都是Python对象,所有对象可以归纳为4种:

  • Tag：标签
  • NavigableString：标签内的文字
  • BeautifulSoup：一个文档的全部内容
  • Comment：注释，特殊类型的 NavigableString 对象

下面用第三方库lxml取代python 3 自带的htmlparser。
